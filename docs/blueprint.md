# **App Name**: CareerCopilot

## Core Features:

- Smart Job Search Engine: Aggregate job postings from multiple sources (LinkedIn, Indeed, Glassdoor, company sites), filter by salary, location, and remote options.
- AI Resume Doctor: Rewrite and optimize resumes using AI, generate cover letters, and check for ATS compatibility.
- Salary Negotiation Coach: Provide data-driven salary insights, negotiation scripts, and company budget ranges to help users negotiate higher salaries. LLM acts as a tool.
- Interview Prep Bot: Conduct role-specific mock interviews, analyze user answers, and predict questions for specific companies. LLM acts as a tool.
- Career Tracker Dashboard: Track job applications, set reminders, and visualize progress through charts.
- Job Posting (Recruiters): Enable recruiters to post jobs with AI candidate matching, diversity hiring tools, and analytics dashboard.
- Company Research Tool: Surface data on culture fit, employee happiness, growth trajectory, and interview difficulty.

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2) for trust and professionalism, fitting the AI focus.
- Background color: Light desaturated blue (#E1F5FE) for a clean, professional feel.
- Accent color: Green (#90EE90) as an analogous hue, used for positive feedback and success indicators within the platform.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Code font: 'Source Code Pro' for displaying code snippets related to technical skills or job requirements.
- Use professional and consistent icons throughout the platform to represent various features and functionalities.
- Maintain a clean, intuitive, and mobile-first layout with clear sections for job search, resume building, and career tracking.
